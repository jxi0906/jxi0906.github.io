Wish List Application


This application is created by Java Servlet to keep track of wish list. Java Servlet is a Java program that runs inside a Web Container. It generates dynamic web content similar to CGI scripts and does not have a main() method in Java program. Additionally, this application will be stored in a client's cookie, and it has 2 buttons: "Add Wish" to store users' record and "Clear All" to clear the existing wish list.
This application contains the main files like WishList.java and HtmlUtil.java. Also, WishList.jpg and screen.css files are used to render and decorate this application.
