Java Club Application


Java Club is constructed as a skeletal structure of a login-based web application using MVC paradigm, Java Servlet, JSP with Expression Language and JavaBean. JSP(JavaServer Pages) technology allows programmer to easily create web content that has both static and dynamic components. MVC architecture is a common design framework for GUI applications, and it is to divide an application into 3 separate components: Model, View and Controller in order to minimize dependencies between them.



Additionally, this application is an online social network club to share programming knowledge and job opportunities among members. It contains a home page to allow to login to the member's main page, or to go to registration form page to sign up "Java Club". When a user registers the signup form successfully, the user's information will be stored in database. When a registered member logins successfully, the user is redirected to the member's main page. Main page will show a greeting message, the member's profile and a logout link.
The application would look like the following.


index.html file as home page of Java Club has a heading and a form to allow to login with user ID and password. "Log in" button requests Login servet, and "Sign up here" link requests Register servlet.



Login.java servlet and Login.jsp allow user to login to the main page. When the login is failed from index.html, the error message will be shown. If a matched user is found in the database, the page will be redirected to MainPage servlet.



Register.java servlet and register.jsp create the signup form. "Sign up" button will trigger Register servlet to take 6 request parameters: user ID, password, re-enter password, first name, last name and email address and validate them using UserDatabase.java, then add to the database. If an input is invalid, the signup form will be re-generated. If a user is successfully added, the page will forward to registerResult.jsp to display a confirmation message.





registerResult.jsp notify user to know the registration was successful or failed. There is a link to home page.



MainPage.java servlet and mainPage.jsp generate the user's main page containing the user main information. There is a link to logout.jsp.



logout.jsp closes the current session and unbind all attributes. There is a "goodbye" message and a link to index.html.
