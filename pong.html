Pong Game


Pong is one of the earliest arcade video games and the very first sports arcade video game. It is a table tennis sports game featuring simple two-dimensional graphics. The game can be played by one player against a computer controlled paddle. The game was originally developed by Allan Alcorn and released in 1972 by Atari corporations. The two paddles return the ball back and forth. The score is kept by the numbers at the top of the screen.
This application contains the main files like Ball.java, Color.java, Paddle.java, Pong.java, SoundPlayer.java, StartPong.java, Vector2.java and TestVector2.java. Also, blip.mav sound file can be used to increase this game's attractiveness.
The libraries of Pong are:

  • gluegen-rt.jar
  • jogl-all.jar
  • jogl-all-natives-windows-amd64.jar
