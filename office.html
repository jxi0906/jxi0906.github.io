Office Location Inventory


This GUI application is created to manage inventory items using JavaFX framework and MVC paradigm. Users can add, edit, and delete records in the created file. This application not only contains some proper date validation when user adds or deletes a record, but also includes the confirmation when deleting a record, even though user can navigate or search all the records in the file.
This application contains the main files like AboutForm.fxml, AboutFormController.java, AddForm.fxml, AddFormController.java, Location.java, LocationInventory.java, LocationInventoryModel.java, MainForm.fxml, MainFormController.java, SearchForm.fxml and SearchFormController.java. Also, location_icon.png file is used to add a icon for this application.
The application would look like the following.





User can click "File" menu and select "Open" button to open a file from user's local computer.



User can click "Save As" button under "File" menu to save a file to user's local computer.



After opening a file from local, user may need to edit the office's address, post code and category, then save or cancel this operation.



User can click "Add record" button under "Edit" menu to add a new location including location ID, address, post code and category.



User can click "Search record" button under "Edit" menu or "Search Locations" button in the "Home" page to search a existing location by its ID or address, then click "Select" button to edit, add or delete location.



When a location is selected, user may delete this location through the button "Delete Location" in the "Home" page or the button "Delete record" under "Edit" menu.



User can check this application's basic information and copyright from the "About" button under "Help" menu.



If users want to quit this application, they can click the "Exit" button under "File" menu or the "Close" button in the top-right corner.
