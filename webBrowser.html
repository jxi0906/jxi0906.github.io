Personal Web Browser


Personal Web Browser Application is created by JavaFX APIs and Scene Builder with MVC paradigm. Users can type a URL address in the textfield, and press the enter key or click "Go" button to load the requested web page on the WebView. The layout can be dynamically stretched or shrunk whenever the window is resized.
This application contains the main files like WebBrowser_junhui.java, MainFormController.java, WebStack.java and MainForm.fxml.
