Clock Widget


Clock Widget is a JavaFX application including JavaFX Scene Builder and UI Controls. JavaFX is the newest GUI framework/API, included since Java SE v7. MVC paradigm (Model, View and Controller) is used in JavaFX application. One of MVC paradigm (controller) is reflected on a new XML-based markup language (FXML) be used, but also web component (WebView) is used to render HTML in JavaFX application. Moreover, JavaFX supports more than 60 UI controls and CSS styling.
This application contains the main files like ClockWidget.java, FXMLDocumentController.java, FXMLDocument.fxml and widget.css. Also, digital-7.ttf file is used to change the clock widget's font and render this application.
